🔧 Configuración requerida

Para que el CRM funcione con tu base de datos real, necesitás reemplazar estos valores en el archivo index.html:

SUPABASE_URL = 'https://TU-PROYECTO.supabase.co' SUPABASE_KEY = 'eyJ...tu-anon-key...'

Encontrás estos datos en: Supabase → Settings → API

También ejecutá el archivo supabase_schema.sql en: Supabase → SQL Editor → New Query
